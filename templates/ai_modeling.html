Genetic Algorithm to Select SP500 Investment Portfolios

Genetic Algorithm

1) Create a chromosome (portfolio) of genes (stocks), one stock per S&P500 sector

2) Build 50 Initial parent portfolios from random selection from the pool of S&P500 composite stocks

3) Construct a fitness function based on investment parameters such as portfolio returns, risk, beta, yields, Sharpe ratio, Treynor measure and Jensen measure

4) Generate a child portfolio by crossover of two parent chromosomes

5) Measure the fitness of chromosome of the child portfolio

6) Replace a parent portfolio by the child portfolio if it has a higher fitness value

7) Apply 3% mutation to chromosomes in each population

8) Continue the crossover, selection and mutation cycle from generation to generation

9) Evaluate portfolios in each generation with fitness function until the best portfolio is shown up or the max number of generations is reached

Market Data

1) Use libcurl and Json to process daily price, fundamental and index for S&P 500 composite stocks

2) Use 10 years (1/1/2010-12/31/2019) of daily price data for each S&P 500 stock and SPY data for portfolio gene and chromosome creation

3) Use 1/1/2020 to 6/30/2020 for portfolio back testing

4) Use 7/1/2020 to present for portfolio probation testing
