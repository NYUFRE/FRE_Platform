Pairs Trading with Unsupervised Machine Learning


This project uses unsupervised machine learning package DBSCAN to find tradeable pairs for pairs trading. The work is based on “Pairs Trading with Machine Learning” by Jonathan Larkins.

Model Building:

1) Calculate the simple daily returns on S&P500 stocks with adjusted close price and use them as the major training source. Increase the complexity of training data by stacking on fundmanental information.

2) Use PCA to reduce the data dimensionality.

3) Use DBSCAN to find the clusters within training data, repeat the process with different parameter &epsilon until the most number of clusters can be found.

Back Testing

Use the adjusted close price over selected time period to conduct pair trading on all possible pairs in each cluster. Calculate the annualized sharpe ratio for each pair and select the top 10 paris.

Pair Trading Logic:

1) Price Ratios: rt = (stock 1 price) / (stock 2 price)

2) MA1t = Rolling mean of r with window1 (30 days)

3) MA2t = Rolling mean of r with window2 (5 days)

4) stdt = Rolling standard deviation of r with window2

5) Z Score: Zt = (MA1 - MA2) / std

6) Open Trades:
    • Open short position if Zt >= threshold, sell stock 1 and buy stock 2
    • Open long position if Zt <= -threshold, buy stock 1 and sell stock 2

7) Close Trades:
    • Close the open positions for each pair at market close

Probatioin Test:

Use the 10 best pairs in the back testing period and carry on the trading til today. Use SPY over the same period as the benchmark.
