Stock Selection


Before constructing a portfolio, stock selection is necessary. The goal of the model is to select the top stock in each sector with the highest predicted sharpe ratio.

Stock select:

A simple LSTM Model is used in this section to predict stock price.

Historical price data are from 2010-01-01 to 2020-12-31, which are used as inputs to predict stock price, then calculate predicted sharpe ratio.

Get the top stock from each sector based on the rank of the predicted Sharpe ratio to generate an equal-weighted sector-neutral portfolio.

Back Test:

Use data from 2021-01-01 to the latest trading day to back test the model to check if the portfolio beat the market.

Three sets of graphs provided
    • Portfolio Daily Return vs S&P 500.
    • Portfolio Cumulative Daily Return vs S&P 500.
    • Individual Top Stock Cumulative Daily Return vs S&P 500.
