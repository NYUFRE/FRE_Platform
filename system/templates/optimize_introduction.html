Portfolio Optimization


Portfolio optimization in finance is the technique of creating portfolio that maximize the investment return and minimize the risk. The key idea of portfolio optimization is based on the Modern Portfolio Theory and the Efficient Frontier. The goal of the model is optimizing returns of portfolio with stocks and bonds while minimizing the risk.

Build Portfolio:

Use data from 2010-01-01 to 2020-01-01 as input data to analysis the optimal portfolio.

Generate four different optimal portfolio using PyPortfolioOpt package.
    • Maximize Sharpe Ratio.
    • Minimize Volatility.
    • Maximize Sharpe Ratio with constraint.
    • Minimize Volatility with Constraint.

Demonstrate 4 portfolios' Return, Volatility, and Sharpe Ratio.
