Earnings Impact Analysis

This is a analysis system where users perform research on how the earnings surprise affect the stock returns.
Users would select a period of time to analyze.

Get Earnings Calendar

1) Get earnings calendar from Yahoo Finance for a period of time.

2) Calculate earnings surprise for each stock:
Surprise % = (Actual EPS – EPS Estimate) ÷ EPS Estimate

3) Sort all the surprises in ascending order, and split all the stocks into 3 groups with relatively equivalent numbers of stocks:
i. Highest surprise group: Beat Estimate Group
ii. Lowest surprise group: Miss Estimate Group
iii. The rest stocks in between: Meet Estimate Group

Retrieve Data From DataBase

1) Retrieve price and return data of SPY500 stocks and SPY in this period of time

2) Define day “zero” for a stock as the day the earning is announced.
Then align the announcement day's data for each record.

Conduct The Bootstrap Analysis

1) Sampling the stocks by randomly selecting 30 stocks from each group, total 90 stocks.

2) For each stock calculate the daily returns Rit for 30 days before the day “zero” and 30 days after: t = -30, -29,...-1, 0, 1,..., 29, 30:
Rit = (Pricet – Pricet-1)/ Pricet-1 (adjusted daily closing price is used for calculation)

3) Calculate the corresponding daily return Rmt for SPY for the same days.
Define abnormal returns as the difference ARit= Rit–Rmt.
Calculate average daily abnormal returns for each group of stocks for all 60 reference days
Cumulate the returns on the first T days to CAAR

4) Repeat steps 1 to 3 30 times and then Calculate:
i. Average AAR and CAAR for 30 samplings for each group
ii. Standard Deviation of AAR and CAAR for each group
