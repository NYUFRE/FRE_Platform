Alpha Test Analysis

This is a factor backtesting system where users could test many famous technical factor.
Factors are backtested throughout the whole period of the database.

Retrieve Data From DataBase

1) Retrieve price and return data of SPY500 stocks and SPY in this period of time

2) Define each day's tommorrow's return as our target to predict.

Technical Indicator Calculation

1) Iterated by each stock to calculate the technical indicator.

2) Pack the open, high, low, close, volume data of each stock in a dict data type.

3) Use the package TA-Lib's build-in method function to calculate the technical indicator.

Technical Indicator Evaluation

1) Evaluation is based on a daily regression. The target is regressed against the indicator each day.

2) Daily single variable regression beta is calculated.
BETAt = COV(xt,yt)/VAR(xt)

3) Daily Information Coefficient(IC)(correlation) is calculated.
ICt = COV(xt,yt) / (Std(xt) * Std(yt))

3) Information Ratio of a period of time is calculated.
It is basically a hypothesis test that daily IC is not zero.
IR = mean(Rhot) / sqrt(VAR(Rhot)/n)

For example, a perfect prediction's Beta and IC are both 1 or -1.
The sign does not matter, and it is about the absolute value.
A bad prediction's Beta and IC are both approximate zero.
In fact a single indicator just have a little prediction power.
A combination of indicators could have good effect.

Evaluation Display

1) All these metrics aggregated by year is displays in a table.

2) Cumulative summation of daily beta is plotted.
In this plot alpha's direction in different time period is showed.

3) Barplot of yearly IC is plotted.
In this plot alpha's yearly stronghood is showed.
