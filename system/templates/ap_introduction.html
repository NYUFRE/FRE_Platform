Assets Pricing Module


The assets pricing module is based on Quantlib package to price financial instrument. Current implemented pricing module are: 1. European Option, 2. American Option, 3. Fixed Rate Bond, 4. Credit Default Swap 5. Forward Rate Agreement, 6. Interest Rate Swap, 7.Plot Yield Curve

European option pricing module:

Calculate European option value and greeks using analytic(Black Scholes) pricing engine, and display graph of y-variable with respect to change in x-variable while holding other variables constant

Note:

1) Assume constant risk free rate, dividend rate and volatility

2) Follow Actual/Actual day convention

3) Using NYSE calendar to determine trading day

Input Parameter Definition:

Spot Price: Current price of the underlying assets

Strike Price: The set price at which the underlying assets can be bought or sold when option is exercised

Day to Expiration: Number of days until the last day that the option is valid

Risk Free Rate: The current risk-free rate or the rate to discount future cash flow

Dividend Rate: Dollar dividend divided by underlying asset price

Volatility: Estimate of the future volatility for the underlying asset

X-axis Parameter: Parameter to display at X-axis

Y-axis Parameter: Parameter to display at Y-axis

American option pricing module

Calculate American option value and greeks using Finite Difference Black Scholes pricing engine, and display graph of y-variable with respect to change in x-variable while holding other variables constant

Note:

1) Assume constant risk free rate, dividend rate and volatility

2) Follow Actual/Actual day convention

3) Using NYSE calendar to determine trading day

4) Vega and Rho currently are not available

5) time grid and space grid are respectively set at 2000 and 200 for computing Finite Difference Black Scholes

Input Parameter Definition:

Spot Price: Current price of the underlying assets

Strike Price: The set price at which the underlying assets can be bought or sold when option is exercised

Day to Expiration: Number of days until the last day that the option is valid

Risk Free Rate: The current risk-free rate or the rate to discount future cash flow

Dividend Rate: Dollar dividend divided by underlying asset price

Volatility: Estimate of the future volatility for the underlying asset

X-axis Parameter: Parameter to display at X-axis

Y-axis Parameter: Parameter to display at Y-axis

Fixed rate bond module:

Calculate fixed rate bond price using discount cash flow pricing engine.

Note:

1) Assume constant risk free rate, dividend rate and volatility

2) Follow 30/360 day convention

3) Using US government bond calendar to determine trading day

Input Parameter Definition:

Face Value: The amount paid to the holder at maturity

Coupon Rate: Annual coupon payments paid by the issuer relative to the bond's face

Risk Free Rate: The current risk-free rate or the rate to discount future cash flow

Valuation Date: The date when to price the bond

Issue Date: the date on which bond begins to accrue interest

Maturity Date: the ending date of the bond

Frequency: Coupon Frequency

Credit default swap module

Calculate Credit Default Swap using mid point CDS pricing engine..

Note:

1) Assume constant discount rate, recovery rate, harzard rate

2) Follow Actual/365Fixed day convention

3) Assume payment is transferred at 20th of payment month

Input Parameter Definition:

Notional Value: The specified amount on which the periodic payments are based

Spread: The percentage of the notional amount that the protection buyer must pay the protection seller over the length of the contract

Recovery Rate: The percentage of notional value that can be recover if default happened

Hazard Rate: The probability of default in a short interval

Risk Free Rate: The current risk-free rate or the rate to discount future cash flow

Issue Date: The start date of the contract

Maturity Date: The ending date of the contract

Frequency: Payment exchange Frequency

Forward rate agreement module

Calculate forward rate agreement using discounting pricing engine..

Note:

1) The underlying index is USD Libor rate

2) The forward rate agreement follows the same day count convention as the USD Libor

3) Continuous compounding for discount factor, but simple compounding for forward rate

4) The yield curve is built based on the data from valuation date

Input Parameter Definition:

Notional Value: The specified amount on which the periodic payments are based

Valuation Date: The "current" date to price FRA

Months to contract start from valuation date: The number of months to contract start date from valuation date

Months to contract termination from valuation date: The number of months to contract start expire date from valuation date

Fixed rate for the forward rate agreement: The rate for the fixed payment.

Interest rate swap module

Calculate interest rate swap using discounting pricing engine..

Note:

1) The underlying index is USD Libor rate

2) The float rate payment and the fixed rate payment exchanged at the same date and at the same frequency

3) No spread on the floating rate

4) float rate payment follows actual/360-day convention and fixed rate payment follows 30/360-day convention

5) The yield curve is built based on the data from valuation date

Input Parameter Definition:

Notional Value: The specified amount on which the periodic payments are based

Swap Start Date: The date when swap contract starts

Payment frequency in month : The number of months between exchanging the payment

Contract period in month: The number of months for the swap contract

Fixed rate: The rate for the fixed payment.
