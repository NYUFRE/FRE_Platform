Fixed Income Trading

This is a fixed income security inquiry and analysis system where a user can search bonds, build portfolios, retrieve advanced analysis data on a given security.

Search Bond

1) List real-time active bond(s) satisfying user's need from market data provider EOD (must upgrade bundle for bond data).

2) Show bond information including ones provided by EOD:
    • Name, Coupon, Yield to Maturity, Maturity Date, Payment Frequency, and (Flat) Price
and ones from analysis based on the information above:
    •, Full Price, Accrued Interest, Macaulay Duration, Modified Duration, and Convexity.

3) Save the bond with a personalized alias.
By default, the bond will be saved with alias as its name.

Build Portfolio

1) Build a hedged portfolio of longing/shorting a main bond of face value set by the user hedged by shorting/longing two other different bonds

2) Regarding the property of portfolio hedging, the maturity data of the bonds selected must follow:
    • Hedged_bond_1 < Main_bond < Hedged_bond_2
    • Hedged_bond_2 < Main_bond < Hedged_bond_1

3) The hedging logic based on the rules:
$$MV_{main} = MV_{hedged1}+MV_{hedged2}$$
$$ModDur_{main} = \frac{MV_{hedged1} \cdot ModDur_{hedged1} + MV_{hedged2} \cdot ModDur_{hedged2}}{MV_{hedged1}+MV_{hedged2}}$$
where MV stands for market value, and ModDur stands for modified duration.

4) Show the calculated parameters of the portfolio including actions, weights, market value, and face value

5) Save the portfolio with a personalized alias.
By default, the portfolio will be saved with combination of the aliases of bonds included.

Yield Change Simulation

1) Simulate changes on price of a bond or security if applying a quantified increase or decrease on the yield to maturity

2) Logic:
The price change of a bond follows:
$$\Delta{Price} = [-(\Delta{Yield}) \cdot ModDur+0.5(\Delta{Yield})^2(Convexity)](FullPrice)/100$$
